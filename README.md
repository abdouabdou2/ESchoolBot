# ESchool Bot — دليل التثبيت

## كيف يعمل النظام كاملاً

```
الطالب يفتح الويب → يدخل اسمه + يرفع إيصال Bankily
        ↓
الذكاء الاصطناعي يقرأ الإيصال ويستخرج: المبلغ + رقم المعاملة
        ↓
ينقر "إرسال" → رسالة واتساب + إرسال للتطبيق على هاتفك
        ↓
إشعار Bankily يصل على هاتفك
        ↓
التطبيق يقارن رقم المعاملة + المبلغ
        ↓
✅ تطابق → يرسل لبوت تلغرام: "طالب جديد | الاسم | المعرف"
        ↓
البوت يرد "تم التسجيل" → التطبيق يُشعرك
```

---

## خطوات التثبيت

### 1. تثبيت التطبيق

افتح المشروع في Android Studio:
```
File → Open → اختر مجلد ESchoolBot
```

ثم شغّل على هاتفك (Run → Run 'app')

### 2. تفعيل إذن الإشعارات

عند فتح التطبيق أول مرة:
- انقر على الـ Switch
- سيفتح إعدادات الإشعارات
- فعّل **ESchool Bot**

### 3. ربط بوت تلغرام

في ملف `NotificationService.java` غيّر:
```java
private static final String BOT_TOKEN = "ضع_token_الجديد_هنا";
```

### 4. رفع صفحة الويب

ارفع ملف `web/index.html` على:
```
https://slip-retainer-phonics.ngrok-free.dev/
```

## ملاحظات مهمة

- **اسم حزمة Bankily**: إذا لم يعمل، افتح هاتفك وتحقق من اسم حزمة تطبيق Bankily
  في `NotificationService.java` السطر: `BANKILY_PACKAGE = "com.bankily"`
  
- **البوت**: تأكد أن البوت استقبل رسالة واحدة على الأقل من حسابك حتى يعرف التطبيق chat_id

- **Token تلغرام**: غيّره من `@BotFather` بعد نشر هذا الكود
